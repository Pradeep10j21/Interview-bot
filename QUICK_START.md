# Quick Start Guide

## Step 1: Install Dependencies
```bash
npm install
```

## Step 2: Start the Application
```bash
npm start
```

The app will automatically open in your browser at `http://localhost:3000`

## Troubleshooting

### If npm install fails:
- Make sure Node.js is installed (v14 or higher)
- Check: `node --version`
- If not installed, download from: https://nodejs.org/

### If the app doesn't open automatically:
- Manually open: http://localhost:3000

### Browser Requirements:
- **Recommended**: Chrome or Edge (best Speech Recognition support)
- Firefox/Safari: Limited Speech Recognition support

### Camera/Microphone Issues:
- Make sure to allow permissions when prompted
- Check browser settings if permissions are blocked

## What to Expect:

1. **Home Page**: Select a company to interview with
2. **Interview Page**: 
   - Click "Start Interview"
   - Allow camera/microphone access
   - Answer questions by speaking
3. **Results Page**: View your performance analysis

## Stop the Server:
Press `Ctrl + C` in the terminal
